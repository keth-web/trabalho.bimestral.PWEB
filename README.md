Descrição da atividade API:
---------------------------
Felipe: Este projeto consiste no desenvolvimento de uma API REST utilizando Node.js e Express, com o objetivo de gerenciar uma lista de tarefas. A API permite realizar operações básicas de um CRUD (Create, Read, Update e Delete), possibilitando criar, visualizar, atualizar e remover tarefas. 

Kethilly: As tarefas são armazenadas em memória e possuem três atributos principais: um identificador (id), um título e um status indicando se estão concluídas ou não. A comunicação com a API é feita através de requisições HTTP, utilizando os métodos GET, POST, PUT e DELETE. Durante o desenvolvimento, foram aplicados conceitos fundamentais de servidores web e APIs REST, como definição de rotas, tratamento de requisições e respostas em formato JSON.
